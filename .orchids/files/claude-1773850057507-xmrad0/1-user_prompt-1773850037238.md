Design and build a modern, high-performance web application called NeonPay – Personal Finance Tracker.

🎯 PURPOSE:
This is a personal finance management app designed to help a single user track income, expenses, analyze spending habits, and receive intelligent financial insights. The app should feel like a premium fintech dashboard, not a basic expense tracker.

🎨 DESIGN STYLE:

Dark mode UI (primary background: deep black #0D0D0D)

Neon green accents (#00FF9F)

Futuristic fintech aesthetic

Clean, minimal layout with strong visual hierarchy

Glassmorphism cards with subtle glow effects

Smooth animations and micro-interactions

Fully responsive (desktop-first, mobile-friendly)

🧭 APP STRUCTURE / PAGES:

Dashboard

Transactions

Analytics

Budget

Goals

Settings

Include a modern sidebar navigation with icons.

💳 DASHBOARD (MAIN PAGE):

Display current balance (auto-calculated)

Show total income (monthly)

Show total expenses (monthly)

Show savings (income – expenses)

Include:

Summary cards (balance, income, expenses, savings)

Monthly spending trend chart (line chart)

Recent transactions list (last 5–10 entries)

💸 TRANSACTION MANAGEMENT:
Allow user to add and manage transactions.

Transaction fields:

Amount

Type (Income or Expense)

Category (Food, Transport, Bills, Shopping, Salary, etc.)

Date

Optional note

Features:

Add transaction form (modal or page)

Edit and delete transactions

Transaction history list

Filters (by category, date, type)

📊 ANALYTICS:
Provide clear financial insights using charts:

Income vs Expenses (bar chart)

Spending by category (pie chart)

Monthly trends (line chart)

Ensure charts are visually appealing and match the neon fintech theme.

🎯 BUDGET SYSTEM:

Allow user to set a monthly budget

Show budget usage (progress bar)

Display remaining balance

Trigger visual warning when close to limit

🎯 SAVINGS GOALS:

Create savings goals (e.g. “Buy Laptop”)

Set target amount

Track saved amount

Show progress visually

🤖 AI-STYLE INSIGHTS (RULE-BASED):
Generate intelligent financial messages based on user data. Examples:

“You spent more on food this week compared to last week”

“Your expenses increased by 20% this month”

“You are close to exceeding your monthly budget”

Display insights on the dashboard in a highlighted card.

🔔 SMART ALERTS:

Overspending warnings

Budget limit alerts

Unusual spending notifications

🧠 DATA HANDLING:

Store data locally (localStorage or simple state) for MVP

Structure code so it can easily integrate with a backend (e.g. Supabase) later

⚙️ TECH REQUIREMENTS:

Frontend: React or Next.js

Styling: Tailwind CSS

Charts: Recharts or Chart.js

Animations: Framer Motion

✨ UX REQUIREMENTS:

Fast and smooth interactions

Clean spacing and typography

Loading states and empty states

Intuitive navigation

Visually polished components

📦 BONUS FEATURES (IF POSSIBLE):

Dark/light toggle

Export transactions (CSV or PDF)

Responsive mobile layout

Search functionality

🎯 FINAL GOAL:
The app should look and feel like a real fintech product (similar to a banking dashboard), with strong UI/UX, smooth animations, and practical personal finance functionality. It should be portfolio-ready and visually impressive.
